Mon

Workflow

Lorsqu'il s'agit de concevoir des sites web, tous mes projets commencent sur Figma, où nous discutons et essayons de trouver un accord sur la conception et la direction du projet sur lequel nous travaillons ensemble. Le choix des couleurs, des polices, des éléments sur les sites, et la structure globale du design sont des éléments capitaux dans la construction de la meilleure expérience utilisateur qu'Internet a à offrir.

I usually work with vanilla HTML, CSS and Javascript for small Front-End project as it allows me to work more freely on the architecture of the project. For bigger Front-End projects, I like to work with React JS alongside Tailwind CSS. Je travaille généralement avec HTML, CSS et Javascript pour les petits projets Front-End car cela permet de travailler plus librement sur l'architecture du projet. Pour les plus gros projets Front-End, j'aime travailler avec React JS et Tailwind CSS.

Mes expériences avec les technologies Back-End sont limités mais j'ai déjà travaillé avec NodeJS et PHP par le passé, et je suis toujours ouvert à l'idée d'approfondir mes compétences en technologie Back-End
